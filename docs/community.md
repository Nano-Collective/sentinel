# Community

Sentinel is built by the [Nano Collective](/collective) — a community-led group building open-source AI tools for the people who use them, not for profit. Whether you want to contribute code, sharpen the rule pack authoring guidance, improve the docs, or just be part of the conversation, there are several ways to get involved.

## Discord

Join our Discord server to connect with other users, ask questions, share rule pack patterns, and get help:

[Join our Discord server](https://discord.gg/ktPDV6rekE)

## Contributing

If you want to contribute to the code, read our [CONTRIBUTING.md](https://github.com/Nano-Collective/sentinel/blob/main/CONTRIBUTING.md) guide for development setup, coding standards, and how to submit your changes.

Note that Sentinel deliberately ships **no rule packs** as product. The best way to contribute to the ecosystem's pack coverage is to publish your own packs in your own repository and share them — the pack format is portable by design.

## GitHub

Head to our [GitHub repository](https://github.com/Nano-Collective/sentinel) to:

- Open issues for bugs or feature requests
- Join discussions with others in the community
- Browse the source code

## Supporting the Project

Sentinel is built by volunteers and offered freely. If the project has been useful to you and you would like to help sustain it, the Nano Collective accepts donations and sponsorship to fund the cost of running the collective and the community fund that pays bounties to OSS contributors.

For donation channels, sponsorship, and the current state of contributor compensation:

- **[Support the Collective](/collective/organisation/support)** — donations, sponsorship, and other ways to help.
- **[Economics Charter](/collective/organisation/economics-charter)** — how the community fund works and how scoped paid bounties are agreed.
